# 特殊文字祭り 

## 通常文字
% パーセント, _ アンダースコア, # ハッシュ, & アンパサンド, $ ドル, { } 波括弧, ~ チルダ, ^ ハット

## URL / パス
- https://example.com/a_b?x=1&y=2#sec
- C:\path\to\file_v1.2\foo%bar.txt

## インラインコード
`a_b_c`, `100%`, `x^2`, `C:\path\foo`

## コードブロック
```text
a_b_c
100%
x^2 + y^2
\LaTeX \textbf{bold}
```

## 数式（$…$ はそのまま）

インライン: $x^2 + y^2 = z^2$

ディスプレイ:

$$
E = mc^2
$$

## 矢印・記号
→：`->`, ⇔：`<->`  
≦：`<=`, ≧：`>=`, ±：`+-`
